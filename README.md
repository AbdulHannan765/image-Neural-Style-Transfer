# Neural Style Transfer using AdaIN

## Overview

This project applies the artistic style of one image to another image while preserving the original content. It uses the Adaptive Instance Normalization (AdaIN) technique and a pre-trained VGG encoder to perform fast neural style transfer.

For example:

- Content Image: A photograph of a city.
- Style Image: A painting by Van Gogh.
- Output Image: The city photograph rendered in Van Gogh's artistic style.

---

## Features

- Upload a content image and a style image.
- Control style intensity using a style strength parameter.
- Generate stylized images in seconds.
- Simple web interface built with Flask.
- Deep learning model powered by PyTorch.
